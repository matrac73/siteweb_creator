# Essences d'Art

Site vitrine premium pour Essences d'Art a Albertville, construit avec Next.js 15, TypeScript, Tailwind CSS, Framer Motion, React Hook Form et Zod.

## Prerequis

- Node.js 18.18 minimum
- npm 9 minimum

## Lancer le projet

```bash
npm install
npm run dev
```

Le site sera disponible sur `http://localhost:3000`.

## Commandes utiles

```bash
npm run typecheck
npm run lint
npm run build
```

## Structure

- `app/` : routes App Router, metadata, sitemap et robots
- `components/` : header, footer, hero, sections, galerie, formulaire
- `data/` : navigation, services, categories Palais des Thes, projets, coordonnees
- `lib/` : helpers SEO, schema.org et utilitaires
